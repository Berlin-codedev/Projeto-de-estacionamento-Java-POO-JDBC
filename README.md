Sistema de Controle e Gestão de DadosDescriçãoEste projeto é uma aplicação desenvolvida em Java focada na implementação de boas práticas de arquitetura de software e lógica de controle. O objetivo central foi estruturar um sistema robusto, utilizando a Orientação a Objetos (POO) para garantir a modularidade e a manutenibilidade do código, simulando processos reais de controle de dados.  Arquitetura e PadrõesA aplicação foi estruturada para separar claramente as responsabilidades, seguindo padrões de projeto consolidados no mercado de desenvolvimento:Padrão DAO (Data Access Object): Utilizado para isolar a camada de acesso a dados da lógica de negócio, permitindo maior flexibilidade e organização.  POO (Programação Orientada a Objetos): Aplicação de encapsulamento, coesão e estruturação de classes para facilitar a evolução e a manutenção do sistema.  Lógica Computacional: Desenvolvimento de algoritmos de processamento de entradas e saídas voltados à integridade e precisão das informações.
