# Projeto-de-estacionamento-Java-POO-JDBC
Projeto de estacionamento criado para estudo no curso de programaçao
Este projeto consiste em um sistema de controle desenvolvido com foco em Orientação a Objetos (POO), visando a separação de responsabilidades e a persistência de dados. O sistema foi projetado para simular o processamento de entradas e saídas de um sistema de controle em ambiente desktop, aplicando boas práticas de arquitetura de software.  FuncionalidadesModelagem Orientada a Objetos: Classes estruturadas com foco em encapsulamento e coesão, garantindo a manutenibilidade do software.  Padrão DAO (Data Access Object): Implementação de camada de persistência para isolar o acesso ao banco de dados, facilitando a troca de tecnologias de armazenamento.  Processamento de Dados: Lógica de processamento de entradas e saídas implementada para garantir a integridade dos dados e o fluxo de controle do sistema.  Tecnologias UtilizadasLinguagem: JavaArquitetura: Padrão DAO, POOPersistência: SQL (JDBC)
