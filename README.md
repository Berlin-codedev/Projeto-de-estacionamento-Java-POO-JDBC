# Sistema de Controle e Gestão de Dados

## Descrição
Este projeto é uma aplicação desenvolvida em Java focada na implementação de boas práticas de arquitetura de software e lógica de controle[cite: 3]. O objetivo central foi estruturar um sistema robusto, utilizando a **Orientação a Objetos (POO)** para garantir a modularidade e a manutenibilidade do código[cite: 3].

## Arquitetura e Padrões
A aplicação foi estruturada para separar claramente as responsabilidades, seguindo padrões de projeto consolidados no mercado:

* **Padrão DAO (Data Access Object):** Utilizado para isolar a camada de acesso a dados[cite: 3].
* **POO (Programação Orientada a Objetos):** Aplicação de encapsulamento e coesão[cite: 3].
* **Lógica Computacional:** Desenvolvimento de algoritmos de processamento de entradas e saídas[cite: 3].

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Banco de Dados:** SQL
* **Ferramentas:** IntelliJ IDEA / Eclipse
